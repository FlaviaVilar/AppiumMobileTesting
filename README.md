# Appium Mobile Testing for Android 

Project covering Appium features automating native Android apps.


## Requisites

- Java
- Android Studio
- Node.js
- Appium
- TestNG

## Setup

- Clone the repository;
- Set appium server and driver for your local configuration on BaseTest.java;
- Run application;
